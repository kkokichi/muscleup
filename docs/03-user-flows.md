# 03. ユーザーフロー

## フロー1: トレーニング記録（最重要・3タップ設計）

```mermaid
flowchart TD
    A[ホーム] -->|タップ1: 記録ボタン| B[ワークアウト記録画面]
    B -->|タップ2: 種目を選ぶ<br>（前回種目が上位に表示）| C[セット入力]
    C -->|前回の重量・回数が自動プリセット| C
    C -->|タップ3: セット完了| D{次のセット?}
    D -->|＋/−で微調整して完了| C
    D -->|セッション終了| E[保存]
    E --> F[マスコットが称賛<br>PB更新なら祝福演出]
    F --> A
```

**ポイント**
- 種目選択リストは「最近使った順」でソート → 常連種目は1タップ目に出る
- セット入力は前回値プリセット + ステッパー → キーボード不要
- 自己ベスト更新はセット完了時に即時判定・即時祝福

## フロー2: 成長を実感する

```mermaid
flowchart TD
    A[ホーム] --> B[進捗タブ]
    B --> C[種目を選択]
    C --> D[重量推移グラフ]
    D --> E[推定1RM / 自己ベスト / 月間成長率]
```

## フロー3: 種目を学ぶ

```mermaid
flowchart TD
    A[部位マップ] -->|部位をタップ| B[部位別の種目一覧]
    B --> C[種目詳細]
    C --> D[フォーム動画・やり方・注意点]
    D -->|この種目で記録する| E[ワークアウト記録画面]

    F[種目辞典] -->|検索 / 部位フィルタ| B
```

## フロー4: 毎日の習慣ループ（継続の核）

```mermaid
flowchart LR
    A[アプリを開く] --> B[マスコットの今日のひとこと]
    B --> C[継続日数・週間回数を確認]
    C --> D{今日はトレ日?}
    D -->|Yes| E[記録する → 称賛 → レベルUP]
    D -->|Rest| F[休養を肯定するメッセージ]
    E --> A
    F --> A
```
