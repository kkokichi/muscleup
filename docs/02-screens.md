# 02. 画面一覧

## MVP画面

| ID | 画面名 | パス | 概要 |
|---|---|---|---|
| S-01 | ホーム | `/` | 今日の予定・最近の記録・週間回数・継続日数・AIコメント・レベル・PB更新数 |
| S-02 | ワークアウト記録 | `/workout/new` | セッション記録。種目追加→セット入力（重量/回数/RPE）→保存 |
| S-03 | 履歴一覧 | `/history` | 過去セッションの一覧（日付降順） |
| S-04 | 履歴詳細 | `/history/[id]` | 1セッションの全セット・メモ・所要時間 |
| S-05 | 進捗（重量推移） | `/progress` | 種目選択→重量推移 / 推定1RM / 自己ベスト / 月間成長率 |
| S-06 | 種目辞典（一覧） | `/exercises` | 検索・部位フィルタ付き種目一覧 |
| S-07 | 種目辞典（詳細） | `/exercises/[id]` | 動画・やり方・効く部位・注意点・初心者ポイント・失敗例 |
| S-08 | 部位マップ | `/muscle-map` | 人体SVG。部位タップ→種目一覧 |

## 共通レイアウト

- **AppShell**: 下部タブバー（ホーム / 記録 / 進捗 / 種目 / マップ）＋ヘッダー
- **記録タブ**は中央に大きいアクセントカラーのFAB風ボタン（最短導線）
