# 12. デザインガイドライン

Apple Human Interface Guidelines を参考に、「シンプル・余白・高級感」を軸とする。

## カラーパレット（ダークモード標準）

| トークン | 値 | 用途 |
|---|---|---|
| `--background` | `#0A0A0B` | 画面背景（純黒に近いが真っ黒は避ける） |
| `--card` | `#161618` | カード背景 |
| `--card-elevated` | `#1E1E21` | 浮いたカード・シート |
| `--border` | `#2A2A2E` | 罫線（低コントラストで控えめに） |
| `--foreground` | `#F5F5F7` | 主要テキスト（Apple風のオフホワイト） |
| `--muted-foreground` | `#8E8E93` | 補助テキスト（HIG systemGray） |
| `--primary` | `#BFFF00` (lime) | アクセント。CTA・グラフ・達成表示 |
| `--primary-foreground` | `#0A0A0B` | アクセント上のテキスト |
| `--destructive` | `#FF453A` | 削除（HIG systemRed dark） |
| チャート第2色 | `#FF9F0A` | 推定1RM線（HIG systemOrange dark） |

- アクセントは**ライムグリーン**を採用（黒基調との相性・エネルギー感）
- 彩度の高い色は面で使わず、**点で効かせる**（ボタン・数値・アイコン）

## タイポグラフィ

- フォント: システムフォントスタック（`-apple-system, "Hiragino Sans", "Noto Sans JP"`ベース）
- 数値（重量・回数・継続日数）は `font-variant-numeric: tabular-nums` + 太字で主役にする
- 見出し: 22-28px / 本文: 15-16px / キャプション: 12-13px
- 行間はゆったり（1.5〜1.6）。**余白で高級感を出す**（カード内 padding 20px 基準）

## レイアウト

- モバイルファースト（max-width 480px の中央寄せ。デスクトップでもアプリ感を維持）
- カードUI基本。角丸 `16px`（HIG continuous corner 風）
- 下部タブバー: 5項目、中央「記録」はライムの円形ボタンで隆起させる
- セーフエリア考慮（`env(safe-area-inset-bottom)`）

## アニメーション（Framer Motion）

| 場面 | 演出 |
|---|---|
| 画面/カード出現 | `opacity 0→1, y 12→0`、stagger 50ms、duration 0.35s、easeOut |
| 数値表示 | カウントアップ（継続日数・1RM） |
| セット完了 | チェックがバウンス（spring） |
| PB更新 | マスコットジャンプ + 紙吹雪 + haptic風スケールパルス |
| タブ切替 | アイコンの微小スケール（0.9→1） |

- **動きは短く控えめに**。0.4s超のブロッキングアニメーション禁止
- `prefers-reduced-motion` を尊重

## コンポーネント規範

- タップターゲット最小 44×44px（HIG準拠）
- 入力はステッパー優先。数値タップでキーボード入力にフォールバック
- 破壊的操作は必ず確認ダイアログ + 赤
- 空状態には必ずマスコット + 次のアクション誘導を表示

## トーン & ボイス（マスコット「マッスー」）

- 明るく短い日本語。絵文字は1つまで。説教しない
- 休養日を肯定する（「休むのも筋トレ！」）— 罪悪感を与えない設計
